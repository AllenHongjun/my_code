-- Inside Microsoft SQL Server 2008: T-SQL Programming
----------------------------------------------------------------------

1. Create a folder called C:\InsideTSQL2008\ and extract the contents of the zip file TSQLProgrammingYYYYMMDD.zip into this folder.

The following files contain the source code for the book:

Chapter 01 - Views.sql
Chapter 02 - User Defined Functions.sql
Chapter 03 - Stored Procedures.sql
Chapter 04 - Triggers.sql
Chapter 05 - Transactions and Concurrency.sql
Chapter 06 - Error Handling.sql
Chapter 07 - Temporary Tables  and Table Variables.sql
Chapter 08 - Cursors.sql
Chapter 09 - Dynamic SQL.sql
Chapter 10 - Working with Date and Time.sql
Chapter 11 - CLR User Defined Types.sql
Chapter 12 - Temporal Support in the Realtional Model.sql
Chapter 13 - XML and XQuery.sql
Chapter 14 - Spatial Data.sql
Chapter 15 - Tracking Access and Changes to Data.sql
Chapter 16 - Service Broker.sql
Appendix A - Companion to CLR Routines.sql

The files in the subfolder VisioFiles are used in some of the code samples in Chapter 13.
The code samples expect the visio files to reside in the folder C:\InsideTSQL2008\VisioFiles\.

2. Run the code in the file InsideTSQL2008.sql to create the book's sample database InsideTSQL2008.
